# キリシマ
格子上に線を引いて領土を奪い合う新型ボードゲーム。  
https://kirishima1020.github.io/kirishima-game/

## 遊び方
- 交互に1手ずつ。1手＝隣り合う格子点を結ぶ1辺を引く（縦横のみ、斜めなし）。
- 各プレイヤーの初手だけは任意。以降は自分の線とつながった点からのみ延ばせる（相手の線は使えない）。
- 未使用の辺をたどって盤の外へ出られなくなった領域は、閉じた手番プレイヤーの色で塗られ確定する（塗り替え不可）。
- 壁は色を問わず数える＝**最後に閉じた者が中の未塗りマスを総取り**。
- **確定した面の中には誰も線を引けない**（塗りは不可侵）。
- 打てる手が無くなった人は飛ばし、**全員が打てなくなったら終了**。塗ったマスが多い方が勝ち。
